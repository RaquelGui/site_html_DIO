Entendendo a diferença entre listas ordenadas e não ordenadas

voltar

As principais diferenças entre listas ordenadas e listas não ordenadas são a forma como os itens são exibidos e a numeração/estrutura aplicada a eles.

Vamos destacar essas diferenças:

Lista ordenada

  • A lista ordenada é criada usando a tag ol.
  • Os itens são organizados em ordem numérica (1, 2, 3, etc.) por padrão, mas você pode usar atributos para personalizar a numeração.
  • Também pode alterar o tipo de marcador usando o atributo type.

Lista não ordenada

  • A lista não ordenada é criada usando a tag ul.
  • Os itens não têm uma ordem específica, e eles são normalmente exibidos com marcadores por padrão (por exemplo, círculos, quadrados ou discos)

    • Ambos os tipos de listas permitem que você coloque qualquer conteúdo dentro dos itens da lista, incluindo texto, imagens, links, etc. A escolha entre usar uma lista ordenada ou não ordenada depende da natureza dos dados que você deseja apresentar e do estilo visual que você quer para a lista.
